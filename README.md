## Important üö®

### 1) Project Type
This project is a **desktop application** designed to automate the car rental process.  
It allows users to manage rentals, view available vehicles, and update rental information.

### 2) Core Logic
The core logic is written in **C language**.  
It handles user input, rental validation, and interaction with the database.

### 3) Database
The application uses **Microsoft Access** for:  
- Initial data verification  
- Storing vehicle and rental information  
- Updating availability status  

### 4) Graphical Interface
The graphical user interface is implemented using **Windows Forms**,  
which is part of the **Microsoft .NET Framework**.

### 5) Correct Launch
To run the application correctly:  
1. Make sure **Microsoft Access** is installed.  
2. Connect the Access database file.  
3. Verify correct database paths in the project.  
4. Build and run the application on **Windows OS**.
